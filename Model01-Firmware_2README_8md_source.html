7 
8 ## Set up the Arduino IDE
9 
10 Setup the Arduino IDE on your system. Make sure you install at least version 1.6, since older version may not support all required features.
11 
12 ### macOS
13 
14 On macOS, install the Arduino IDE using [homebrew](http://brew.sh/) [cask](https://caskroom.github.io/) with `brew cask install arduino` or download the application from [the official website](https://www.arduino.cc/en/Main/Software) and move it to your `/Applications` folder.
15 
16 
17 ### Linux
18 
19 On Linux, your distribution's package manager probably includes the Arduino IDE, but it may be an out of date version. If your distribution installs a version of Arduino before 1.6.3 or so, you'll need to install the Arduino IDE following the instructions [on the wiki](https://github.com/keyboardio/Kaleidoscope/wiki/Arduino-Setup-Linux).
20 
21 If you install Arduino into some place that's /not/ `/usr/local/arduino`, you'll need to set the `$ARDUINO_PATH` environment variable to the directory containing the `arduino` and `arduino-builder` binaries.

54 # Build and flash the firmware
55 
56 Before you begin, make sure your Model 01 is connected to your computer.
57 
58 ### Option 1: From the command line
59 
60 ```sh
61 cd Model01-Firmware
62 make flash
63 ```
64 
65 When the builder tells you to hit Enter to continue, hold down "Prog" in the top left corner of your keyboard and hit Enter.
66 
67 ### Option 2: From the Arduino IDE
68 
69 
70 Open the sketch you wish to flash (for example, `Model01-Firmware.ino`).
71 
72 Click the Upload button or press `Ctrl-U`.
73 
74 Hold down the "Prog" key in the top left corner of your keyboard, until the compile finishes and the upload begins.

77 # Start to customize the firmware
78 
79 You can customize your keyboard's key layout and LED effects by modifying the `Model01-Firmware.ino` file in the same directory as this README. Model01-Firmware.ino is a computer program written in 'Arduino C'. You can find documentation about Arduino C at https://arduino.cc.

82 # Updating the firmware and libraries
83 
84 From time to time, you may want to pull the latest version of this firmware from GitHub. To do that, navigate to the Model01-Firmware directory in your shell and pull the latest code.
85 
86 ```sh
87 cd Model01-Firmware
88 git pull
89 ```
