Detailed Description

Sometimes there are situations when one wants to ignore key events for a while, to mask them out. Masked keys will be ignored until they are released.

This is implemented in the Hardware library because that knows best how to mask efficiently, as this requires a deeper knowledge of the hardware, which is all but hidden from the rest of the plugins.
