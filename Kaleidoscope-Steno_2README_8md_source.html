12 Stenography is a way to write in shorthand, a chorded input system that allows
13 very fast input (considerably higher than normal touch typing), by using
14 shorthand chords and a dictionary. This plugin implements the `GeminiPR`
15 protocol that supports a number of systems, including [Plover][plover].
16 
17  [plover]: http://www.openstenoproject.org/plover/
18 
19 While Plover supports a normal QWERTY keyboard too, having a dedicated plugin
20 comes with important advantages:
21 
22 * No need to toggle Plover on and off, because the normal keys are not taken
23  over by Plover anymore.
24 * Easier toggling, because you only have to toggle the layer, not Plover too. If
25  you switch back to a keyboard layer, without toggling Plover off, nothing
26  unexpected will happen. Plover will not take over the keys.
27 * The `GeminiPR` protocol supports language systems other than English.
28 
29 All of this comes at a cost, however: the protocol is implemented over the
30 virtual serial port, any plugin that wants to use that port too, will run into
31 conflicts with the Steno plugin. In other words, don't use it together
32 with [Focus][k:focus].
33 
34  [k:focus]: https://github.com/keyboardio/Kaleidoscope-Focus

36 ## Using the plugin
37 
38 To use the plugin, simply include the header in your Sketch, tell the firmware
39 to use the `GeminiPR` object, and place Steno keys on your keymap. It is best
40 illustrated with an example:
41 
42 ```c++
43 #include <Kaleidoscope.h>
44 #include <Kaleidoscope-Steno.h>
45 
46 // Somewhere in the keymap:
47 S(S1), S(S2), etc
48 
49 void setup() {
50  Kaleidoscope.use(&GeminiPR);
51 
52  Kaleidoscope.setup ();
53 }
54 ```
55 
56 ## Keys provided by the plugin
57 
58 The plugin provides a number of keys one can put on the keymap, that allow
59 correspond to various Steno keys. All of these must be used together with the
60 `S()` macro provided by the plugin, as can be seen in the example above.
61 
62 The provided keys are: `FN`, `N1`, `N2`, `N3`, `N4`, `N5`, `N6`, `S1`, `S2`,
63 `TL`, `KL`, `PL`, `WL`, `HL`, `RL`, `A`, `O`, `ST1`, `ST2`, `RE1`, `RE2`, `PWR`,
64 `ST3`, `ST4`, `E`, `U`, `FR`, `RR`, `PR`, `BR`, `LR`, `GR`, `TR`, `SR`, `DR`,
65 `N7`, `N8`, `N9`, `NA`, `NB`, `NC`, `ZR`.
66 
67 ## Plugin methods and properties
68 
69 The plugin provides a `GeminiPR` object, with no public methods or properties.
