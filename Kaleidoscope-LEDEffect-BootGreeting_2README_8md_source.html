9 If you want to have your keyboard signal when it turns on, but you don't want to
10 use any more complicated LED modes, this plugin is for you. It will make the
11 `LEDEffectNext` key on your keymap slowly breathe for about ten seconds after
12 plugging the keyboard in (without blocking the normal functionality of the
13 keyboard, of course).
14 
15 ## Using the plugin
16 
17 To use the plugin, include the header, and tell `Kaleidoscope` to use the plugin:
18 
19 ```c++
20 #include <Kaleidoscope.h>
21 #include <Kaleidoscope-LEDEffect-BootGreeting.h>
22 
23 void setup() {
24  Kaleidoscope.use(&BootGreetingEffect, &LEDOff);
25 
26  Kaleidoscope.setup();
27 }
28 ```
29 
30 The plugin provides no methods or properties, the above is all it can do.
