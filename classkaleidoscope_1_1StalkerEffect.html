void kaleidoscope::StalkerEffect::onActivate ( void  )
finalprotectedvirtual

Function to call whenever the mode is activated.

Like setup, this method need not touch LEDs, update will be called right after it. The purpose of this callback is to allow a plugin to do some preparation whenever it is activated, instead of only on boot, or always at each cycle.

However, unlike setup, this method can change LED colors, if so desired. Either to provide an initial state, or a static color set. In the latter case, consider implementing refreshAt too, because other plugins may override some of the colors set at activation time, and refreshAt can be used to restore them when needed.

Before the callback runs, LEDs will be blanked.

Refresh the color of a given key.

If we have another plugin that overrides colors set by the active LED mode (either at time, or via update), if that plugin wants to restore whatever color the mode would set the key color to, this is the method it will call.

Parameters
